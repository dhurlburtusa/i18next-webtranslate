# Introduction

This is the web translation UI to i18next.

For more information go [here](http://i18next.com/pages/ext_webtranslate.html)

# Developers

To build the solution:

1. `npm install grunt -g`
2. `grunt release`

During development:

1. `node server.js`
2. `grunt watch` or `grunt debug
